Privacy & telemetry

Pluks is local-first, but anonymous usage data and crash reports help us fix bugs and prioritize what to build next. Here's exactly what we do.

  • Password fields are skipped automatically. The desktop app checks macOS Accessibility before every capture — if focus is inside a secure text input (login, unlock prompt, password manager), Pluks never simulates a copy.
  • What we send: anonymous event counts (panel opens, history clicks, smart-paste use), error and crash stack traces, app/OS version, locale.
  • What we never send: the text you copy, the websites you visit, your name or email, anything that could identify you.
  • How to turn it off: desktop app → ⚙ Preferences → "Send anonymous usage stats". Website → respects Do Not Track.
