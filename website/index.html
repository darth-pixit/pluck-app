How to install (Chrome / Edge / Brave)

  1. Download pluks-extension.zip and unzip it.
  2. Open chrome://extensions and enable Developer mode (top-right toggle).
  3. Click Load unpacked and select the unzipped folder.
  4. The Pluks ⚡ icon appears in your toolbar — done!

Firefox

  1. Download and unzip pluks-extension.zip.
  2. Open about:debugging#/runtime/this-firefox.
  3. Click Load Temporary Add-on and select manifest.json inside the unzipped folder.
